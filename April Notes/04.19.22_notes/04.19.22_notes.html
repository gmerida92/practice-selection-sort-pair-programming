My HTML Cheat Sheet

Headings

Headings work by using h1, h2, and h3 tag. Only use h1 when necessarily then in ascending order (as each header number grow it become less important). Header with lower number are more optimize for SEO.

Paragraph are used for when we have full sentences.

  • Unordered lists (ul) are list using bullet points instead of numbers or numerical system.
  • Ordered lists (ol) are list using number or numerical system. It does not use bullets.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute Go to DuckDuckGo to search for more information

Setting `target="_blank"` will open the link in a new window. See MDN for more details

Images

We can use the img tag to add images to your page.

Tables

Tables in HTML are great for displaying tabular data, such as an address book or a list of product description and prices. Tables should only be used for displaying data, not for defining the layout of a page.

Tags that can be useful but not as widely used as above-written tags

  • article elements:

    • represents a self-contained composition in a document, page, application, or site,
    intended to be independently distributable or reusable

  • footer elements:

    • represents a footer for its nearest sectionining content or sectioning root element
    Typically contain information about the author, and any other related copyright related data

  • header elements:

    • represents introductory content, typically a group of introductory or navigational aids.
    May contains some heading elements but also a logo, a search form, an author name, and other elements

  • nav element:

    • represents a section of a page whose purpose is to provide navigation links, either within
    the current document or to other documents.
